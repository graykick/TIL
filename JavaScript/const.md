# const
```const```는 ES6에 추가된 문법이다.
이는 변수를 선언할 때 사용된다.
기존에는 ```var```키워드만 존재했지만, ES6부터는 ```let```과 ```const```가 추가되었다.

## const는 상수이다.
그래서 재정의가 불가능하다.
여기서 재정의가 불가능 하다는 말은 한번 할당된 값을 변경할 수 없다는 것이다.
